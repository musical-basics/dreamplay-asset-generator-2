# DreamPlay Asset Generator

## AI-powered product asset generator for DreamPlay Pianos

Generates on-brand photos and videos for all major platforms using Google's Gemini, Imagen 4, and Veo 3.1.

### Setup
1. Get a free Gemini API key at [aistudio.google.com](https://aistudio.google.com)
2. Copy `.env.example` to `.env.local` and paste your key
3. Run `pnpm install && pnpm dev`

### Output Platforms
- Instagram, Meta Ads, Reddit Ads
- Website Heroes (desktop + mobile)
- Shopify product images & banners
- YouTube, TikTok, Google Display Ads

### Models Used (Free Tier)
- **Imagen 3** — product & lifestyle photos
- **Veo 2** — short video clips & reels
- **Gemini 2.0 Flash** — prompt enhancement & reference analysis
